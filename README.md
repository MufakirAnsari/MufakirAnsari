## Featured Projects
#### 🧬 [DDFF-Framework](https://github.com/MufakirAnsari/DDFF-Framework)
Distributional Discriminative Feature Filtering for HDLSS biological datasets. Four complementary divergence metrics (L1, L2, L∞, symmetric KL) for interpretable biomarker discovery. Includes benchmarking, runtime profiling, and biological validation.
#### 🧫 [Ebola RNA-seq Pipeline](https://github.com/MufakirAnsari/ebola-viral-rnaseq)
End-to-end HPC pipeline processing 356 Ebola outbreak SRA runs on SLURM at Ohio Supercomputer Center. Dual quantification with HISAT2 + Kallisto. 14 publication-grade figures.
#### 🛡️ [MiniMe](https://github.com/tryminime/minime)
Privacy-focused AI platform spanning backend, website, graph systems, docs, infra, and deployment. Multi-repo product architecture built for scale and privacy.
#### ⚡ [Dy-Part](https://github.com/MufakirAnsari/Dy-Part)
Dynamic, noise-aware scheduler for hybrid quantum-classical algorithms. Systems optimization for emerging compute architectures.
#### 🏥 [Clinical-Grade Cancer Detection](https://github.com/MufakirAnsari)
SimCLR pretraining + ResNet50 pipeline over 277K+ pathology image patches. AUC-ROC 0.950, FNR as low as 0.34%. Grad-CAM, UMAP, t-SNE explainability.
